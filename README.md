# E-Commerce-Sales-Analysis
### Project Overview
This project explores sales performance for an E-Commerce store by analyzing revenue, customer behavior, product performance, and sales trends.I built an interactive Excel dashboard that helps uncover insights such as top-selling products, monthly revenue patterns, and customer purchasing behavior.

### Data Sources

Customer Data:The primary dataset used for this analysis is the "customers_data.csv" file, containing detailed information about each customer that patronise the company.

### Results

1. Card Transaction showed higher volume compared to other transaction type
2. Beverages-Energy/Protein is the top selling product
3. Sales peaked during specific months.
4. Some regions consistently underperformed, showing market improvement potential.
### Recommendation
Based on the analysis, we recommend the following actions:
 - Offer targeted promotion to Poland and Germany to improve sales
 - Invest more in card marketing channels due to higher transaction volume
 - Improve stock levels for fast-moving products
 

### Limitation
- Dataset is fictional and may not represent real world complexities 
- Basic Conditional Logic
- No Temporal Analysis
